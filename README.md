# Image Privacy Checker

A browser extension that removes sensitive metadata like geolocation from images before uploading onto the web, enhancing user privacy
and data security.

## Usage 

* Open Google Chrome and type `chrome://extensions/` in the address bar.
* Toggle Developer mode (top right).
* Click on Load unpacked and select the folder containing the extension files.
* After the extension is loaded, click the extension icon in the toolbar and drag and drop / select your image in the popup box.
* Your image will be stripped of location metadata and uploaded into the webpage automatically.

## Features

* Removes location metadata from the image before upload.
* Uploads the image directly into gmail, Google Drive, and other websites.
* Simple, easy interface to drag and drop or select images.  

<br>

**NOTE:**  Currently works only on Google Chrome.
